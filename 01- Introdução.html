/* FUNÇÕES → Sempre que eu preciso reutilizar um código, eu uso uma função. function sum(Parâmetro da função, podem ter até 30 parâmetros) { console.log() } sum(argumentos) // INVOCA OU CHAMA A FUNÇÃO (INVOKE OR CALL). RETURN: Quando quer retornar o resultado de uma função, fora dela. ANONIMA: Inserir uma função dentro de uma variavel. ARROW: Reduz espaço, diminui o tamanho do código. ARRAY - METÓDOS → Push: adiciona novos itens ao array; → Length: mostra o tamanho, quantos itens tem no array; → Sort: organiza o array em ordem alfabetica; → Delete: deleta um item do array; → Splice (unir/ ligar): coloca ou tira itens extamente da posição que queremos tirar; → Slice (fatiar): selciona qual/ quais itens quer mostrar; → Pop: remove o ultimo item do array; → Shift: remove o primeiro item do array; FOR EACH É por onde se calcula a média de algum array. MAP → Cria um novo array, a partir do array percorrido (array original); → Cria um novo array, com a quantidade de itens do array original; → Aceita 3 parâmetros: - item do array; - index; - array completo. FILTER → Cria um novo array, a partir do array percorrido (array original); → Cria um novo array APENAS com elementos FILTRADOS; → Aceita 3 parâmetros: - item do array; - index; - array completo. REDUCE → Retorna um valor (pode ser um novo array, um objeto, string, number, etc.) → Aceita 4 parâmetros: - acumulador; - valor atual; - index; - array completo. */
